How do I test whether an element has a particular class?

.hasClass() handles this common use case:

$( "div" ).click(function(){ if( $(this).hasClass("protected")){ $(this) .animate({ left: -10}) .animate({ left: 10}) .animate({ left: -10}) .animate({ left: 10}) .animate({ left: 0}); } });

You can also use the .is() method along with an appropriate selector for more advanced matching:

if ($("#myDiv").is(".pretty.awesome")){ $("#myDiv").show(); }

Note that this method allwos you to test for other things as well. For example, you can test whether an elemnt is hidden ( be using the custom :hidden selector) :

if( $("#myDiv").is(:hidden)){ $("#myDiv").show(); }
